# Calx-AI
AI-powered calorie estimation system using food recognition and adaptive nutrition analysis.

Project Overview

Version 1 goal:

User searches for a food item and receives nutrition information.

---

Roadmap

- V1 → Text search + nutrition lookup
- V2 → Image recognition
- V3 → Portion estimation
- V4 → Adaptive depth estimation

---

Version 1 Architecture

User
↓
FastAPI Backend
↓
USDA API
↓
Nutrition Response

---

Tech Stack

- Python
- FastAPI
- USDA FoodData API

---

Completion Criteria

- User can search food
- Nutrition data is returned
- Errors are handled
- Project pushed to GitHub
